# Test Project Setup

### Setup:
```
git clone https://github.com/sfmoe/grdm.git
```
```
cd grdm
```
```
npm install
```

Rename ".env.sample" to ".env" and update urls and api keys to correct values.

if you dont enter new values the server will use the local sample files.

-------
### Run Server:

```
npm run start
```

### View Project:

http://localhost:3000
